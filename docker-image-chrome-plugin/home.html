帮助

先看这几条，再排查复杂问题。

怎么开始下载

打开 Docker Hub 的 tags 页面，找到目标 tag 和架构，点击行右侧下载按钮。下载结果是 tar 包，可直接 `docker load -i image.tar`。

什么时候要配置认证

只有私有镜像、受限仓库，或者匿名拉取触发限流时，才需要补 Docker Hub 账号或访问令牌。

为什么会失败

最常见的是 `401`、`429`、代理节点不可用、或目标 tag/架构不存在。失败原因会直接显示在任务卡片里。
